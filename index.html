Virtual box y una iso de la distribucion de ubuntu

Actualizacion so -- apt update

Creacion usuario root -- sudo -i

Cambio contrasena -- passwd root

instalacion paquetes ssh -- apt install ssh

Generacion de llave SSH -- ssh-keygen -b 4096

Confirmacion ip e instalacion paquetes de herramientas de red -- ifconfig y net-tools

instalacion apache -- apt install apache

configuracion puerto salida pagina 8088 -- vi /etc/apache2/apache2.conf

modificacion informacion muestreo pagina publicada -- vi /var/www/html/index.html

Para conocer los detalles del servidor existen diversas formas como la instalacion de herramientas neofetch, htop o top -- apt install neofetch

Y mediante comandos como df -h free -h lsb-release -d ifconfig
